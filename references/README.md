# References

This directory is reserved for stable reference material used by
`ai-skill-governance`.

References must be:

- human-readable,
- auditable,
- directly relevant to skill governance,
- sourceable when they summarize external material,
- and small enough to load intentionally.

Do not add large reference corpora, generated dumps, hidden memory, RAG-oriented
content, or retrieval infrastructure unless the operational need is explicit and
documented first.

The absence of additional references is acceptable. This directory exists as a
governed extension point, not as a requirement to accumulate material.
